<h1> Searching Algorithms</h1>

<p>
  These algorithms are designed to check for an element or retrieve an element from any data structure where it is stored.<br>
  The following Algorithms will be covered : 
</p>

<ol>
      <li>Linear Search </li>
      <li>Binary Search</li> 
      <li>Ternary Search</li>
</ol>
</div>

<div class="linear-search" >
  <h2>1. Linear Search Algorithm </h2>
  <p>Sequential searching algorithm which iterate over set of elements present in the array until the target is found . <br>
     If Found:  returns the element with the index m otherwise false <br>
   Implementation Can be found here : 
   <a href="LinearSearch.java">Linear Search</a> 
  <br>
    Time complexity: O(N).
    Auxiliary Space: O(1).
</p>

</div>

<div class="binary-search" >
  <h2>2. Binary Search Algorithm </h2>
  <p> Implemented in a sorted array by repeatedly dividing the search interval in half.   <br>
      Implementation Can be found here :   <a href="BinarySearch.java">Binary Search</a> 
   <br>
   Time Complexity: O(log n)
   Auxiliary Space: O(1)
</p>
</div>

<div class="ternary-search" >
  <h2>3. Ternary Search Algorithm </h2>
  <p> Implemented in a sorted array by repeatedly dividing the search interval in three.   <br>
      Implementation Can be found here :   <a href="TernarySearch.java">Ternary Search</a> 
   <br>
 Time Complexity: O(log3n)
 Auxiliary Space: O(log3n)
</p>
</div>
