<h2> Sorting Algorithms </h2>

<p>
  A Sorting Algorithm is used to rearrange a given array or list of elements according to a comparison operator on the elements. 
  The comparison operator is used to decide the new order of elements in the respective data structure.
</p>

<div class="bubble-sort">
  <h2>Bubble Sort</h2>
  <p>
   Bubble sort is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order. <br>
   The algorithm gets its name from the way smaller elements "bubble" to the top of the list. <br>
   The procedure is repeated as many times as necessary, until the list is sorted. <br>
   for Implementation Check Out !: <a href="BubbleSort.java">Bubble Sort</a> <br>
   Time Complexity: O(N^2) <br>
   Auxiliary Space: O(1) 
  </p>
</div>

<div class="insertion-sort">
  <h2>Insertion Sort</h2>
  <p>
   Insertion sort is a simple sorting algorithm that builds the final sorted array one item at a time. <br>
   It is much less efficient on large lists than more advanced algorithms such as quicksort, heapsort, or merge sort. <br>
   The basic idea is to iterate through the list, compare the current item with the items on the left of it, 
   and swap them until the current item is in its correct position relative to the previous items. <br>
   for Implementation Check Out !: <a href="InsertionSort.java">Insertion Sort</a> <br>
   Time Complexity: O(N^2)  <br>
   Auxiliary Space: O(1)
  </p>
</div>

<div class="merge-sort">
  <h2>Merge Sort</h2>
  <p>
   Merge Sort is a divide-and-conquer based sorting algorithm that works by recursively dividing the unsorted list into n sub-lists until each 
   sub-list has only one element and then merging those sub-lists in a manner that results into a sorted list. <br>
   It is a comparison-based algorithm <br>
   for Implementation Check Out !: <a href="MergeSort.java">Merge Sort</a> <br>
   Time Complexity: O(N log(N))  <br>
   Auxiliary Space: O(n) 
  </p>
</div>
